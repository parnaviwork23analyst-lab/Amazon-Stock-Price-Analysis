# Amazon Stock Price Analysis

Exploratory Data Analysis (EDA) on Amazon historical stock market data using PySpark.

## Features
- Data loading using PySpark
- Summary statistics
- Missing value analysis
- Closing price trend visualization
- 7-day moving average
- Open, High, Low price analysis
- Price volatility analysis
- Correlation analysis
- Monthly average price analysis
- Volume vs Close price correlation

## Technologies
- Python
- PySpark
- Pandas
- Matplotlib
- Jupyter Notebook
